# SmartResume-Generator-Customized-Resumes-for-Every-Opportunity
SmartResume Generator is a Streamlit-based web application that dynamically generates a professional, ATS-friendly resume in PDF format based on user-provided details.

## Overview
The application collects structured resume information such as personal details, career objective, education (school, intermediate, and college), academic projects, internships, technical skills, and certifications. Based on the inputs, it generates a clean and well-aligned resume with section-wise separators.

## Key Features
- Interactive web interface using Streamlit
- Structured education input (school, intermediate, college)
- Support for unlimited academic projects
- Professionally formatted PDF resume
- ATS-friendly layout with clear alignment and section separators
- Runs on a local server (localhost)

## Technology Stack
- Python
- Streamlit (UI)
- FPDF (PDF generation)

## Project Type
Web Application (Localhost)
